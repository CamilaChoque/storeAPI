# store API
La manipulación de la API [fakestoke](https://fakestoreapi.com/) a modo de práctica y adquirir conocimiento hacia las operaciones de _agregar_ , _obtener_ , _eliminar_ .

### Instalación de dependencias :wrench:
```
npm install
```
Se recomienda en el package.json agregar `"type":"module"` para que funcione el uso de funciones de otro archivo al index.js

### Empezar a probar :nut_and_bolt:
```
npm start GET products
npm start GET products/1
npm start DELETE products/1
npm start POST products T-shirt-Rex 300 remeras
```
